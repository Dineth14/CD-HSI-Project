Status: Completed · Dates: 20/04/2026 – 26/04/2026

Planned Work

  • Literature review on change detection techniques
  • Identify gaps in existing methods
  • Identifying Datasets and understanding Image features

Findings

Observations

  • Currently various techniques are being used for change detection and semantic segmentation in remote sensing images. CNN and transformer based models have evolved and shown promising results.
  • Datasets such as DSIFN-CD, WHU-CD, SYSU-CD, LEVIR-CD are commonly used for benchmarking change detection algorithms.
  • Change class limitations and dataset biases need to be addressed for more robust model performance.

Decisions

  • Identify characteristics of datasets using classical image processing methods.
  • Focus on addressing change class limitations and dataset biases in the proposed method.
