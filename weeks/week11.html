Week 11 — Decoder, Refinement Head, and Loss Design

Status: Completed · Dates: 05/07/2026 – 11/07/2026

Planned Work

  • Design the Adaptive Receptive-Field (ARF-FPN) Decoder for multi-scale region feature fusion.
  • Implement the Bounded Boundary Residual Refinement head to refine coarse change logit predictions.
  • Formulate the multi-task joint loss function combining BCE, Dice, coarse, and Sobel-edge boundary supervision.
  • Evaluate model variants (A3, A4, A5) on benchmark datasets to quantify incremental performance gains.

Completed Work

E. Adaptive Receptive-Field Decoder

The decoder operates on the CRAM-lite refined region stream. Since CRAM-lite is applied only to the first three scales, we set R4 = R4. The decoder input is {Rℓ}ℓ=14. Each scale is processed by an adaptive receptive-field block with parallel dilated depthwise convolutions:

d ∈ {1, 2, 4, 8} (8)

These branches give the decoder both local detail and wider spatial context. The multi-scale ARF-FPN block fuses the four region features in a top-down manner, and the coarse prediction head produces:

Pc = hcoarseARF-FPN({Rℓ}ℓ=14) (9)

where Pc is the coarse change logit map. This map is the region-level prediction before boundary correction.

F. Bounded Boundary Residual Refinement

The refinement branch uses the selected finest-scale boundary feature B1, the coarse logit Pc, and an edge cue from the coarse prediction:

Ec = Sobel(σ(Pc)) (10)

In the refinement block, Ec is the Sobel block applied to the coarse logits after sigmoid. The boundary aggregation block concatenates the boundary feature, coarse logit, and coarse edge cue:

ΔP = href([B1, Pc, Ec]) (11)

The refinement head predicts a residual correction rather than a new mask. The final logit is:

Pf = Pc + 0.1 · tanh(ΔP) (12)

The tanh bounds the residual and the scale factor keeps the update small, so boundary evidence refines the coarse prediction without overwriting it. The final probability map is:

Ŷ = σ(Pf) (13)

which is thresholded to obtain the binary change map.

G. Training Objective

The full model is trained with final, coarse, and boundary supervision:

L = LBCE + LDice + 0.4 Lcoarse + 0.1 Lboundary (14)

BCE and Dice supervise the final prediction Pf, while the coarse loss supervises Pc to stabilize the region decoder. The boundary term applies a Sobel-edge ℓ1 loss to encourage contour consistency. The auxiliary weights keep the main BCE and Dice objective dominant, and ablations without full supervision disable the coarse and boundary losses.
